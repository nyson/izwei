		setTags (line 123)
	
 
	
	

setTags sets the tags we want to limit our search with. $include is  either an array or a string containing tags we want in our search and  $exclude is an array or string containing tags we want to exclude


example: setTags(array(cute, cuddly_bears), "belch_demons") will set our  search to return all rows with the tags 'cute' and 'cuddly_bears' but  will ignore all rows with the tag 'belch_demon'
example: setTags(null, "nice_things") is a instruction to how you can't  have nice things
